# DesafioJava

Este repositório contém uma coleção de desafios de programação em Java, organizados por níveis e desafios individuais. Cada desafio é uma pequena aplicação que resolve um problema específico, ajudando a praticar conceitos básicos de programação em Java.

## Estrutura do Projeto

- **Nivel1/**: Pasta contendo os desafios do nível 1.
  - **Desafio_001/**: Antecessor de um número
  - **Desafio_002/**: Percentuais de votos em eleição
  - **Desafio_003/**: Reajuste salarial de funcionário
  - **Desafio_004/**: [Descrição não disponível]
  - **Desafio_005/**: Conta bancária
  - **Desafio_006/**: Produto
  - **Desafio_007/**: Leitura de nomes de pessoas
  - **Desafio_008/**: Média de notas de turma
  - **Desafio_009/**: Dados
  - **Desafio_010/**: Username
  - **Desafio_011/**: Utilitários
  - **Desafio_012/**: Programa geral
  - **Desafio_013/**: Half Entry e Ticket
  - **Desafio_014/**: Sistema de vendas (Attendant, Manager, etc.)
- **pdf das questões/**: Arquivos PDF com as descrições detalhadas dos desafios.

## Pré-requisitos

- Java JDK (versão 8 ou superior)
- Um editor de código ou IDE como VS Code, IntelliJ IDEA, ou Eclipse

## Como Executar um Desafio

1. Navegue até a pasta do desafio desejado (ex: `Nivel1/Desafio_001/src`).
2. Compile os arquivos Java:
   ```
   javac application/Program.java entities/*.java
   ```
3. Execute o programa:
   ```
   java application.Program
   ```

Para desafios com estrutura diferente (como Desafio_013 e Desafio_014), verifique os arquivos `Main.java` ou similares.
